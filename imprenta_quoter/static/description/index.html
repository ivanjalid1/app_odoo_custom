From book specs to Sales Order in one click

Replace your quoting spreadsheet with a structured, ORM-backed model that computes paper, plates, machine passes, finishing, folding, gathering, sewing, binding, cutting, packaging, freight, GGFF / profit / commission markup, IGV and final unit price automatically. On approval, one click generates the Sales Order and Manufacturing Order with the exact same components.

  • No double entry. Quote components flow into the Manufacturing Order unchanged.
  • Auditable formulas. Every section is computed and stored.
  • Multiple run sizes. Up to 5 alternatives per quote with auto-recalculated unit price.
  • l10n_pe compatible. IGV 18%, SPOT detractions, SIRE export.
